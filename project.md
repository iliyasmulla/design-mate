# DesignMate Construction Platform

A premium platform that connects **homeowners with trusted construction professionals** such as architects and designers.

---

# Target Customers

## Clients
- House Owners

## Professionals
- Architects
- 3D Visualizers
- Interior Designers
- Structural Engineers
- Landscape Designers

---

# Core Product Vision (Mobile-First)

Make it easy for homeowners to **post construction projects** and get **qualified professionals to apply**, compare bids, and hire the right expert — optimized for the mobile experience.

---

# Core Features (MVP)

### 1. Post Project
Homeowners can post their project with:
- budget
- location
- project description
- images

### 2. Browse Projects
Professionals can browse available projects and filter by:
- budget
- location
- category

### 3. Apply to Project
Professionals submit bids to projects.

### 4. Select a Bid
Clients review proposals and choose the best professional.

### 5. Chat
In-app messaging between client and professional.

### 6. Architect Rating System
Simple **5-star review system** for architects.

### 7. Razorpay Payment Integration
Secure payment handling.

Clients use the platform **for free**.

---

# Verification System

### Client Verification
- OTP based phone verification
- Phone number must be unique

### Professional Verification
- OTP phone verification
- Unique email
- Portfolio link required
- LinkedIn profile link
- Instagram profile link

Platform team verifies portfolios before assigning projects.

---

# Platform Quality Control

When a client posts a project:

1. Architects apply
2. Platform team reviews their portfolio
3. Best architect is shortlisted
4. Project is assigned

This ensures **quality and trust on the platform**.

---

# Monetization Model

### Commission Based Model

Instead of subscriptions, the platform earns **commission per successful project**.

Example:

- Projects **below ₹50,000 → 7% commission**
- Projects **above ₹50,000 → 6% commission**

Commission is paid by the professional who wins the project.
